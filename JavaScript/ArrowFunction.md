## 화살표 함수 기본
화살표 함수란 기존의 함수 표현을 보다 단순하게 만들 수 있는 방법을 의미한다.  
특별한 내용은 아니지만, 기본적인 내용이라 꼭 한 번은 다뤄 보고 싶어서 기록을 남겨놓는다 !

- ### 화살표 함수 사용 X 
```
    let sumb =  function (a,b) {
        return a+b;
    };
```

- ### 화살표 함수 사용 
1. Parameter가 1개
```
    let double = n = n*2;
```

2. Parameter가 2개 이상
```
    let function = (a,b) => a+b;
```
